# 🦊🍜🍥 FANARUTO 🍥🍜🦊

## 1️⃣ Página escogida
El proyecto consiste en una **página web de fans de Naruto Uzumaki**, protagonista del anime *Naruto*. La página incluye historia, personajes principales, curiosidades y hasta un video embebido sobre noticias recientes.

---

## 2️⃣ Identificación de partes semánticas
Se utilizaron etiquetas semánticas de HTML para organizar el contenido:

- `<header>` → Encabezado con el título principal.  
- `<nav>` → Menú de navegación con enlaces a secciones.  
- `<section>` → Bloques de contenido como *Historia* y *Personajes*.  
- `<article>` → Noticias independientes sobre el mundo ninja.  
- `<aside>` → Curiosidades y datos adicionales.  
- `<footer>` → Pie de página con créditos.  

---

## 3️⃣ Justificación del uso de etiquetas
- **`<header>`**: Da identidad y presenta el título de la página.  
- **`<nav>`**: Mejora la accesibilidad y navegación.  
- **`<section>`**: Divide el contenido en áreas temáticas claras.  
- **`<article>`**: Permite mostrar información independiente (ejemplo: noticias).  
- **`<aside>`**: Añade contexto extra sin interrumpir el flujo principal.  
- **`<footer>`**: Cierra la página con créditos y año.  

---

## 4️⃣ Partes eliminadas y razones
- Se eliminaron `<div>` sin propósito semántico para mantener el código más limpio.  
- Se evitó repetir imágenes o texto innecesario, priorizando la claridad y accesibilidad.
